## 2. Area-Ratio Census

Ratio = drawn_map_area / (recorded_cultivable + pot_kharaba).
Plots with ratio ∈ [0.7, 1.4] → placement problem (fixable).
Plots outside → area/record disagreement → investigate/flag.
